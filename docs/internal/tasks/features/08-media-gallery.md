# Фича: Галерея Изображений с Метаданными и Переходом в Чат

## Цель

Сделать отдельную вкладку-галерею для всех изображений (загруженных и сгенерированных) с датой, источником, промптом и быстрым переходом к исходному месту в диалоге.

## Почему это важно для оценки

- Улучшает UX хранения и повторного использования контента.
- Делает мультимодальность практически полезной.
- Усиливает продуктовую целостность и "рабочее пространство" в одном окне.

## Как фича связана с общим сценарием

Пользователь открывает галерею -> находит нужное изображение -> видит промпт/метаданные -> переходит к исходному сообщению в чате -> продолжает работу с контекстом.

## Где работать

- Backend:
  - Индекс медиа-артефактов и метаданных.
  - Связь изображения с сообщением/чатом/промптом.
  - API фильтрации и поиска.
- Frontend:
  - Вкладка галереи с сеткой и фильтрами.
  - Просмотр изображения и карточки метаданных.
  - Навигация обратно в исходный чат и сообщение.

## План действий

- [ ] Определить структуру метаданных медиа-объекта.
- [ ] Реализовать индекс и выборки для галереи.
- [ ] Собрать UI списка, просмотра и фильтрации.
- [ ] Добавить переход "из галереи в точку диалога".
- [ ] Обеспечить единый стиль с интерфейсом чата.

## Риски и зависимости

- Риск: рассинхрон галереи и истории сообщений.
- Зависимость: единая модель сообщений и вложений.
- Зависимость: корректное хранение промптов и связей с сообщениями.

## Критерий готовности для демо

- Галерея показывает изображения с датой и метаданными.
- Из карточки изображения есть переход в исходный чат.
- Пользователь может быстро найти нужное изображение без ручного скролла чата.
