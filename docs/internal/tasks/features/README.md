## Правила использования карточек

- В каждой карточке фиксируем только план действий и зоны ответственности.
- Низкоуровневые реализации ведем в отдельных технических задачах и PR.
- Если фича не доведена до рабочего состояния, не показываем ее в продукте как активную.
- Готовые к демо фичи обязательно переносим в `GPTHub шаблон фич.xlsx`.
