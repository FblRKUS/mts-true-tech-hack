# Фича: OCR Фотографий с Выделением Текста

## Цель

После загрузки изображения дать пользователю возможность выделить область и извлечь текст, который можно отправить в чат как отдельный контекст.

## Почему это важно для оценки

- Усиливает мультимодальность практическим сценарием.
- Делает работу с изображениями полезной, а не только визуальной.
- Хорошо демонстрируется в видео и презентации.

## Как фича связана с общим сценарием

Пользователь загружает фото документа/экрана -> выделяет нужный фрагмент -> OCR извлекает текст -> агент использует этот текст в ответе с учетом памяти.

## Где работать

- Backend:
  - OCR-контур обработки изображений.
  - API для получения текста по выбранной области.
  - Связь OCR-результата с исходным сообщением.
- Frontend:
  - Просмотр изображения с инструментом выделения области.
  - Панель предпросмотра OCR-текста перед отправкой в чат.
  - История OCR-действий внутри диалога.

## План действий

- [ ] Определить UX выделения области и подтверждения OCR.
- [ ] Реализовать API OCR на фрагмент изображения.
- [ ] Добавить в чат вставку распознанного текста как структурированного блока.
- [ ] Поддержать повторное OCR с другой областью.
- [ ] Проверить качество на разных типах изображений.

## Риски и зависимости

- Риск: низкая точность OCR на шумных фото.
- Зависимость: единый storage изображений и метаданных.
- Зависимость: мультимодальная модель сообщений.

## Критерий готовности для демо

- Пользователь выделяет область на фото.
- OCR возвращает текст, который отправляется в чат.
- Ответ агента использует извлеченный текст.
