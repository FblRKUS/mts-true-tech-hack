# Frontend-контракты

## Интеграция

- Frontend общается только с backend API.
- Типы фронтенда синхронизированы с backend-схемами.
- Авторизация и обработка ошибок централизованы.

## Продуктовые зоны UI

- `chat` (лента, композер, ветки)
- `attachments` (файлы, превью, OCR)
- `models` (auto/manual, текущая модель)
- `memory` (история и факты)
- `gallery` (медиа)

## UX-правила

- Единый сценарий для текста/голоса/фото/файлов.
- Пользователь видит режим `Auto/Manual` и выбранную модель.
- Долгие операции имеют прозрачный статус.
